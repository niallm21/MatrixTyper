LIBERTYCLONE

A tiny 3D open-world joyride.

  • Move: joystick / WASD
  • Hijack / exit a car: ENTER button / F — works on moving traffic
  • Attack (on foot): ATTACK button / Space — knock heads clean off
  • Brake / handbrake (driving): BRAKE button / Space
  • Camera: CAM button / C
  • Grab the glowing cash, cause a little chaos.

Drive responsibly. Or don't. It's a sandbox.
